HTML Table Reference Guide

Table Tags

TagNameDescription
<table> Table The wrapper tag for HTML tables.
<thead> Table Head The wrapper for the rows defined as the table's headers.
<tbody> Table Body The wrapper for the rows containing the main table content.
<tr> Table Row The container for a table row.
<th> Table Heading The wrapper for data cells in a table's header.
<td> Table Data The wrapper for a each data cell in a table.
<tfoot> Table Foot The wrapper for the rows defined as the table's footer.

Table Attributes

TagNameDescription
<colspan> Column Span Defines how many columns a data element should span.
<rowspan> Row Span Defines how many rows a data element should span.
